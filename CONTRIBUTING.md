# Contributing to ParkNow

Thank you for your interest in contributing to **ParkNow**! 🚗

## How to Contribute

### 1. Reporting Bugs
- Open an issue describing the bug, including steps to reproduce, expected behavior, and your device environment (Android / iOS / Metro version).

### 2. Feature Requests
- Feature ideas are welcome! Open a feature request issue describing the use case and proposed UI/UX workflow.

### 3. Code Contributions
1. Fork the repository and create a feature branch: `git checkout -b feature/amazing-feature`
2. Ensure clean code format: `npm run lint`
3. Commit your changes with conventional commit format: `git commit -m "feat(module): description"`
4. Push to your branch and open a Pull Request against `main`.

## Code Style & Guidelines
- We use ESLint and Prettier for code formatting.
- Follow React Native best practices: keep components modular, use Zustand for global state management, and isolate Firebase network logic in `src/services/`.

Thank you for helping make urban parking seamless!
